# Release Notes

## Version 2.0.0 (Next)
**Released: In Development**

###  Features
- Complete support for Mintlify docs.json schema (2024-2025)
- Support for 40+ configuration properties
- 7 modern themes: mint, maple, palm, willow, linden, almond, aspen
- 14 analytics providers integration
- AI contextual menu with ChatGPT, Claude, and Perplexity
- Internationalization support for 17 languages
- Flexible navigation (simple, grouped, tabbed, versioned, multi-language)

###  Improvements
- Full JSON Schema validation
- TypeScript definitions
- Smart defaults and best practices
- Migration support from mint.json to docs.json

###  Documentation
- Comprehensive schema reference
- Migration guide
- CLI commands reference
- API documentation
