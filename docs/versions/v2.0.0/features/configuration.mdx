# Configuration

Comprehensive guide to all configuration options available in Mintlifier.

## Core Settings

### Site Information

**Name** (required)
```json
{
  "name": "Your Documentation"
}
```
The name displayed in the header and browser title.

**Favicon** (required)
```json
{
  "favicon": "/favicon.svg"
}
```
Icon shown in browser tabs. Supports `.svg`, `.png`, `.ico` formats.

## Visual Customization

### Themes

Choose from three built-in themes:

```json
{
  "theme": "venus"  // "venus" | "quill" | "prism"
}
```

- **Venus**: Modern, clean design with vibrant accents
- **Quill**: Classic documentation style
- **Prism**: Bold and colorful

### Layout Styles

```json
{
  "layout": "sidenav"  // "topnav" | "sidenav" | "solidSidenav"
}
```

- **topnav**: Navigation at the top
- **sidenav**: Collapsible side navigation (default)
- **solidSidenav**: Always-visible side navigation

### Corner Style

```json
{
  "rounded": "default"  // "default" | "sharp"
}
```

### Colors

#### Basic Colors
```json
{
  "colors": {
    "primary": "#0D9373",
    "light": "#07C983",
    "dark": "#0D9373"
  }
}
```

#### Background Colors
```json
{
  "colors": {
    "background": {
      "light": "#FFFFFF",
      "dark": "#0F1117"
    }
  }
}
```

#### Gradient Anchors
```json
{
  "colors": {
    "anchors": {
      "from": "#0D9373",
      "to": "#07C983"
    }
  }
}
```

### Logo Configuration

#### Single Logo
```json
{
  "logo": "/logo.svg"
}
```

#### Dual Mode Logos
```json
{
  "logo": {
    "light": "/logo-light.svg",
    "dark": "/logo-dark.svg",
    "href": "https://yoursite.com"
  }
}
```

## Navigation Structure

### Navigation Groups

```json
{
  "navigation": [
    {
      "group": "Getting Started",
      "pages": [
        "introduction",
        "installation",
        "quickstart"
      ]
    },
    {
      "group": "Advanced",
      "pages": [
        "configuration",
        {
          "group": "Nested Group",
          "pages": ["nested-page-1", "nested-page-2"]
        }
      ]
    }
  ]
}
```

### Top Navigation Tabs

```json
{
  "tabs": [
    {
      "name": "Documentation",
      "url": "docs"
    },
    {
      "name": "API Reference",
      "url": "api"
    },
    {
      "name": "Release Notes",
      "url": "changelog/release-notes"
    }
  ]
}
```

### Anchors (External Links)

```json
{
  "anchors": [
    {
      "name": "GitHub",
      "icon": "github",
      "url": "https://github.com/yourusername/yourrepo"
    },
    {
      "name": "Discord",
      "icon": "discord",
      "url": "https://discord.gg/yourinvite"
    }
  ]
}
```

## API Documentation

### OpenAPI Integration

#### Single Spec File
```json
{
  "openapi": "/openapi.json"
}
```

#### Multiple Spec Files
```json
{
  "openapi": [
    "/api-v1.json",
    "/api-v2.json",
    "/internal-api.yaml"
  ]
}
```

### API Configuration

```json
{
  "api": {
    "baseUrl": "https://api.yoursite.com",
    "auth": {
      "method": "bearer"  // "bearer" | "basic" | "key"
    },
    "playground": {
      "mode": "show"  // "show" | "simple" | "hide"
    }
  }
}
```

## Features

### Feedback Options

```json
{
  "feedback": {
    "thumbsRating": true,
    "suggestEdit": true,
    "raiseIssue": true
  }
}
```

### Search Configuration

```json
{
  "search": {
    "prompt": "Search documentation...",
    "location": "side"  // "side" | "top"
  }
}
```

### Mode Toggle

```json
{
  "modeToggle": {
    "default": "dark",  // "light" | "dark"
    "isHidden": false
  }

```

## Versioning

### Simple Versioning

```json
{
  "versions": ["v2.0.0", "v1.0.0", "v0.9.0"]
}
```

### Advanced Versioning

```json
{
  "versions": [
    "v2.0.0",
    {
      "name": "v1.0.0",
      "url": "docs/v1.0.0"
    },
    {
      "name": "v0.9.0",
      "url": "docs/v0.9.0"
    }
  ]
}
```

## Configuration Tips

1. **Start Simple**: Begin with required fields and add features as needed
2. **Use Defaults**: Mintlifier provides sensible defaults for most options
3. **Test Locally**: Always preview with `mint dev` before deploying
4. **Validate**: Run `mint validate` to check your configuration
5. **Version Control**: Keep your `mint.json` in Git for tracking changes
