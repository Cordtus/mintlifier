# Introduction to Mintlifier v2.0

Mintlifier is a powerful CLI tool for creating and managing Mintlify documentation configurations with full support for the latest **docs.json schema (2024-2025)**.

<Note>
  Version 2.0 brings complete support for the new Mintlify docs.json format, replacing the legacy mint.json with enhanced features and better navigation structure.
</Note>

## Key Features

### 🎨 Modern Themes
Choose from 7 new themes: `mint`, `maple`, `palm`, `willow`, `linden`, `almond`, `aspen`

### 📊 Analytics Integration
Support for 14 analytics providers including GA4, PostHog, Mixpanel, and more

### 🌍 Internationalization
Built-in support for 17 languages with flexible navigation structures

### 🔧 Complete Schema
- JSON Schema validation
- TypeScript definitions
- Comprehensive documentation

## Quick Start

```bash
# Create a new configuration
npx mintlifier

# Edit existing configuration
npx mintlifier --edit

# Run locally
cd mintlify-docs
npx mint@latest dev
```

## Why Mintlifier?

1. **Interactive Experience**: Step-by-step wizard guides you through all configuration options
2. **Smart Defaults**: Pre-configured with best practices
3. **Validation Built-in**: Ensures your configuration is always valid
4. **Schema Compliance**: Automatic `$schema` reference for IDE support
5. **Migration Support**: Easy upgrade from mint.json to docs.json

## Architecture

Mintlifier is built with:
- Node.js 18+ for modern JavaScript features
- ES modules for clean imports
- Inquirer for interactive prompts
- Chalk for beautiful terminal output
- Full TypeScript type definitions
