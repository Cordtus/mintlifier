# Installation

Get Mintlifier up and running in just a few steps.

## Prerequisites

Before installing Mintlifier, ensure you have:

- **Node.js** (v16 or higher)
- **Yarn** or **npm** package manager
- **Git** (for version control features)

## Installation Methods

### Method 1: Clone from GitHub (Recommended)

```bash
# Clone the repository
git clone https://github.com/Cordtus/mintlifier.git

# Navigate to the directory
cd mintlifier

# Install dependencies with Yarn
yarn install

# Or with npm
npm install
```

### Method 2: Direct Download

Download the latest release from [GitHub Releases](https://github.com/Cordtus/mintlifier/releases) and extract the archive.

```bash
# Extract and navigate
tar -xzf mintlifier-latest.tar.gz
cd mintlifier

# Install dependencies
yarn install
```

## Verify Installation

Run Mintlifier to verify it's working correctly:

```bash
# Using Yarn
yarn start

# Using Node directly
node index.js

# Or make it executable
chmod +x index.js
./index.js
```

You should see the interactive configuration prompt:

```
 Mintlifier - Mintlify Configuration Builder

? Documentation site name: █
```

## Install Mintlify CLI

To preview and deploy your generated documentation, you'll also need the Mintlify CLI:

```bash
# Install globally with npm
npm i -g mintlify

# Or with Yarn
yarn global add mintlify

# Verify installation
mintlify --version
```

## Troubleshooting

### Permission Errors

If you encounter permission errors during global installation:

```bash
# macOS/Linux
sudo npm i -g mintlify

# Or use a Node version manager like nvm
nvm use 16
npm i -g mintlify
```

### Module Not Found

If you see module errors, ensure all dependencies are installed:

```bash
# Clear cache and reinstall
rm -rf node_modules yarn.lock
yarn install
```

### Windows Compatibility

On Windows, use PowerShell or Git Bash for best compatibility:

```powershell
# PowerShell
node index.js

# Git Bash
./index.js
```
